# Troubleshooting

---

## Login Issues

**Can't log in to the Portal**
Make sure you're using the correct organizational email. If you see a message that your organization could not be found, contact your [App] administrator.

---

## Startup Issues

**"Could not connect to [App] Services" message**
Ensure your device is connected to Wi-Fi.

**Device can't recognize the QR code**

- Make sure the device camera is pointed directly at the QR code
- Ensure the entire QR code is visible and not cut off
- Try displaying the QR code on a larger screen, or print it at a larger size

---

## Anchor Issues

**The Anchor is in the wrong position**

Work through these checks in order:

1. Verify the Anchor is locked — there should be no floating sphere visible and no ambient audio
2. Make sure the device's cameras are not obscured. Walk around the room to improve spatial mapping — the Anchor may jump into place on its own
3. If the Anchor is wrong on **one device only**, use the [Anchor Menu](user-guide/anchor-menu.md) on that device to reposition
4. If the Anchor is wrong on **many or all devices**, the Anchor position may need to be reset — see [Advanced Features](advanced/advanced-features.md)
5. If the issue persists, try switching to a Manual Anchor type — see [Anchor Types](advanced/advanced-features.md#anchor-types)

---

## Content Issues

**An object isn't loading**

- Make sure [App] is open on your device and you're in the correct room
- If content isn't loading for any user, verify the Anchor Code was entered correctly
- Large models may take time to load depending on Wi-Fi speed and device capabilities — if you see a loading indicator, wait
- If no loading indicator appears after a while, close the [App] app completely, reopen it, and rejoin your room

**Can't move an object**

- Verify that the content is supposed to be movable — not all objects can be moved
- Ensure all device permissions have been accepted
- Try quitting and restarting the [App] app
- If the object cannot be moved by multiple users and is disrupting the session, your administrator can use Asset Sender to remove all objects from the room

---

## General Device Issues

If a problem is isolated to a single XR device, close the app completely and rejoin. If that doesn't help, fully shut down the device (press and hold the power button), wait 15–30 seconds, then power it back on.
