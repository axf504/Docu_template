# Sending Content to XR

Once your room is set up and your Anchor is placed, you're ready to send content to your XR devices.

---

## Find your Anchor Code

Your Anchor displays a **4-letter code** next to it in XR. You'll need this to connect apps to your device.

- Each device sees a different code, but all codes work
- Enter this code when prompted by any [App] application

---

## Open an App from the Portal

[App] includes pre-built applications you can control directly from your browser.

1. From the Portal homepage, select **Apps**
2. Click any application to open its controller in a new tab
3. Enter your **4-letter Anchor Code** in the field provided
    - On most apps, this is a box in the top-right corner labeled **"None"** until a code is entered
4. The app is now connected — content you send from the browser will appear on your XR device

---

## Tips

- If content doesn't appear, double-check that your Anchor Code was entered correctly
- Large models may take time to load depending on your Wi-Fi speed and device
- If you see a loading indicator, wait — if nothing happens after a while, close and reopen the [App] app and rejoin your room
- Each user in a room can have a different Anchor Code displayed — any of them will work
