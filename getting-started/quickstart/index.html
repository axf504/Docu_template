    Quick Start Guide

    This guide walks a room administrator through setting up [App] for the first time. One person per room completes this setup.

    Step 1: Sign In to the [App] Portal

    On your computer, go to the [App] Portal.

    1. Click Sign In in the top right corner
    2. Choose your sign-in method:
      • Use your organizational email address
      • If your organization uses Google SSO, select Continue with Google
    3. After signing in, you'll see the Portal homepage with three options: Rooms, Meetings, and Apps

    Step 2: Create a Room

    1. Select Rooms → Create Room
    2. Enter your room information:
      • Name (required): Match this to the physical location, e.g. "Conference Room B" or "Anna's Home Office"
      • Description (optional)
    3. Click Next

    Create your Anchor

    Anchors help XR devices align content in your physical space.

    1. Click Add Anchor
    2. Leave the Name field blank — [App] will auto-name it "Center"
    3. Set Type to Automatic
    4. Leave Is Center checked
    5. Leave Anchor Groups blank
    6. Click Add Anchor in the bottom right to confirm

    Save your Room

    1. Confirm you see "Anchor Name: Center" and "Type: Automatic" in the summary
    2. Click Next, then Finish
    3. A dialog will appear with your Room Name, a 6-digit room code, and a QR code

    Save your Room Code

    You'll need the QR code or 6-digit code to connect XR devices to this room. We recommend printing the QR code and posting it in the room for easy access.

    Step 3: Connect your XR Device

    1. Start up your XR headset and make sure the cameras are not obscured
    2. Walk around the room for 10–15 seconds so the device can orient itself to the space
    3. Launch the [App] app from your device's home menu
    4. Accept any permission prompts (camera, microphone)
    5. Connect to your Room using one of these methods:
      • Scan QR Code — point the device camera at the QR code from Step 2 (HoloLens, Mobile)
      • Enter Room Code — use the number pad in XR to type the 6-digit code (all other devices)

    Step 4: Position your Anchor

    The first device to join a new room sets the Anchor position for everyone. Only one device needs to complete this — additional devices should skip to Joining from Additional Devices.

    See the full Room Setup guide for detailed Anchor placement instructions.

    You're Ready!

    Once your Anchor is placed, you can start sending content to XR. See Sending Content to XR to get started.
